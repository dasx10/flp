2. [Array](../README.md)

## DESCRIPTION

Create a new array filled with the given value.

## USE

```javascript
import fillAll from "flp/Array/fillAll/index.js";
```

## EXAMPLES

```javascript
fillAll(1)([1, 2, 3]); // [1, 1, 1]
fillAll("a")(["a", "b", "c"]); // ["a", "a", "a"]
```
