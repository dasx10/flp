## USE

```javascript
import append from "flp/Array/append/index.js";
```

## EXAMPLES

```javascript
var numbers = [1, 2, 3, 4];
append(5)(numbers); // [1, 2, 3, 4, 5];
append(6)(numbers); // [1, 2, 3, 4, 6];
append(7)(numbers); // [1, 2, 3, 4, 7];
```
