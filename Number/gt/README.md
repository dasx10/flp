## DESCRIPTION

Is greater then or equals two numbers. Is equal to `a >= b`


## USE

```javascript
import gt from "flp/p/Number/eq/index.js";"
```

## EXAMPLES

```javascript
gt(1, 2); // false
gt(2, 1); // true
gt(1, 1); // false
gt(2, 2); // false
```
