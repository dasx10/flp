# MAX

1. [Procedures](../README.md)
    2. [Number](../README.md)

## DESCRIPTION

Get the maximum of two numbers.

## USE

```javascript
import max from "flp/p/Number/max/index.js";"
```

## EXAMPLES

```javascript
max(1, 0); // 1
max(0, 1); // 1
max(1, 1); // 1
max(0, 0); // 0
max(1, 2); // 2
max(2, 1); // 2
max(2, 2); // 2
max(0, 2); // 2
max(2, 0); // 2
```
