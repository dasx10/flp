## DESCRIPTION

Is greater then or equals two numbers. Is equal to `a >= b`


## USE

```javascript
import ge from "flp/p/Number/eq/index.js";"
```

## EXAMPLES

```javascript
ge(1, 2); // false
ge(2, 1); // true
ge(1, 1); // true
ge(2, 2); // true
ge(0, 2); // false
```
