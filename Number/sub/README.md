# Sub

1. [Procedures](../README.md)

## DESCRIPTION

Power two numbers.


## USE

```javascript
import sub from "flp/p/Number/sub/index.js";
```

## EXAMPLES

```javascript
sub(3, 2); // 1
sub(2, 1); // 1
sub(1, 1); // 0
sub(2, 2); // 0
sub(0, 2); // -2
sub(2, 0); // 2
sub(0, 0); // 0
sub(-1, 1); // -2
sub(-2, 2); // -4
sub(-3, 3); // -6
sub(-4, 4); // -8
sub(-5, 5); // -10
```
