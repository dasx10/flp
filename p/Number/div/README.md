## DESCRIPTION

Divide two numbers.


## USE

```javascript
import div from "flp/p/Number/div/index.js";"
```

## EXAMPLES

```javascript
div(1, 2); // 0.5
div(2, 1); // 2
div(1, 1); // 1
div(2, 2); // 1
div(0, 0); // Infinity
div(1, 0); // Infinity
div(0, 1); // 0
div(1, -1) // -1
```
