1. [Procedures](../README.md)
    2. [Number](../README.md)

## DESCRIPTION

Add two numbers.


## USE

```javascript
import all from "flp/p/Number/add/index.js";
```

## EXAMPLES

```javascript
add(0, 0); // 0
add(0, 1); // 1
add(1, 0); // 1
add(1, 2); // 3
add(2, 1); // 3
add(1, 1); // 2
add(2, 2); // 4
add(NaN, 1); // NaN
add(1, NaN); // NaN
```
