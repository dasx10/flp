## DESCRIPTION

Power two numbers.


## USE

```javascript
import pow from "flp/p/Number/pow/index.js";
```

## EXAMPLES

```javascript
pow(0, 0); // 0
pow(1, 0); // 1
pow(0, 1); // 0
pow(1, 1); // 1
pow(1, 2); // 1
pow(2, 1); // 2
pow(2, 2); // 4
pow(0, 2); // 0
pow(2, 3); // 6
```
