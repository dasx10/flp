## DESCRIPTION

Average two numbers. Is equal to `(a + b) / 2`


## USE

```javascript
import avg from "flp/p/Number/avg/index.js";"
```

## EXAMPLES

```javascript
avg(1, 2); // 1.5
avg(2, 1); // 1.5
avg(1, 1); // 1
avg(2, 2); // 2
avg(0, 2); // 1
avg(2, 0); // 1
```
