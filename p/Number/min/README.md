# MIN

1. [Procedures](../README.md)
    2. [Number](../README.md)
        3. [min](./README.md)

## DESCRIPTION

Get the minimum of two numbers.


## USE

```javascript
import min from "flp/p/Number/min/index.js";"
```

## EXAMPLES

```javascript
min(0, 1); // 0
min(1, 0); // 0
min(0, 0); // 0
min(1, 1); // 1
min(1, 2); // 1
min(2, 1); // 1
min(2, 2); // 2
min(0, 2); // 0
min(2, 0); // 0
```
