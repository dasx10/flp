## DESCRIPTION

Is equal two numbers. Is equal to `a === b`


## USE

```javascript
import ne from "flp/p/Number/ne/index.js";"
```

## EXAMPLES

```javascript
ne(1, 0); // true
ne(0, 1); // true
ne(1, 1); // false
ne(0, 0); // false
ne(1, 2); // true
ne(2, 1); // true
ne(2, 2); // false
```
