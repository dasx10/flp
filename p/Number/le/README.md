## DESCRIPTION

Less than or equal to two numbers. Is equal to `a <= b`


## USE

```javascript
import le from "flp/p/Number/le/index.js";"
```

## EXAMPLES

```javascript
le(1, 2); // false
le(2, 1); // true
le(1, 1); // true
le(2, 2); // true
```
