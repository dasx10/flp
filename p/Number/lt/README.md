## DESCRIPTION

Less than two numbers. Is equal to `a < b`


## USE

```javascript
import lt from "flp/p/Number/lt/index.js";"
```

## EXAMPLES

```javascript
lt(1, 2); // false
lt(2, 1); // true
lt(1, 1); // false
lt(2, 2); // false
```
